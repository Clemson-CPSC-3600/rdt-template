# rdt-template

## Setup Instructions

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Copy starter code to src directory:
   ```bash
   cp template/*.py src/  # Copy all Python files from template to src
   ```

4. Run tests:
   ```bash
   python run_tests.py
   python run_tests.py -v
   python run_tests.py --bundle 1
   ```
